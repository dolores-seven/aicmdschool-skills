# TED Speech Training Skills

> TED 演讲训练技能库 — 帮助青少年准备有影响力的演讲

---

## 技能列表

| 技能 | 功能 | 触发词 | 适用年龄 |
|------|------|--------|----------|
| **youth-aited-mentor** | 青少年AI主题TED演讲教练，通过苏格拉底式提问引导准备演讲，或优化已有演讲稿 | "/aited"、"TED演讲准备"、"青少年演讲" | 10-17岁 |

## 快速开始

将对应 skill 目录复制到你的 Claude Code skills 目录下即可使用：

```bash
# Windows
cp -r youth-aited-mentor ~/.claude/skills/

# macOS/Linux  
cp -r youth-aited-mentor ~/.claude/skills/
```

## 设计理念

这个技能库的核心理念是：

- **拒绝代写** — AI 代写的演讲稿没有灵魂，我们只做引导和优化
- **苏格拉底式提问** — 通过提问帮助青少年从自己的经历中提炼观点
- **保留原话** — 最好的金句来自青少年的真实表达，不是成人化的修辞

## 演讲力公式

$$
\text{演讲力} = \text{个人叙事(故事)} \times \text{认知冲突(思辨)} + \text{技术洞察(素养)}
$$
