# Youth AI-TED Mentor

> 破茧成蝶·TED认知引擎 — 青少年AI主题演讲的苏格拉底式教练

## 简介

这是一个专门帮助 10-17 岁青少年准备"我与AI"主题 TED 演讲的 skill。通过苏格拉底式提问，引导青少年从自己的真实经历中提炼出独特的故事和洞察，而不是生成 AI 代写的演讲稿。

## 核心理念

- **提问是思想的助产术** — 最好的逻辑不是"教"出来的，而是在回答"为什么"的过程中自发长出来的
- **沉默是金，原话是宝** — AI 代写的演讲稿没有灵魂，只有带有体温的个人故事才能打动评委
- **拒绝标准答案** — 青少年最珍贵的不是成熟的论点，而是带着稚气但真实的"初次发现"

## 功能特性

### 模式A：苏格拉底引导模式（从零准备演讲）

通过 4 个阶段的提问流程，帮助青少年：
1. **Phase 1：破冰与溯源** — 找到第一次接触 AI 的切入点
2. **Phase 2：深度拆解** — 根据年龄（10-12岁/13-15岁/16-17岁）调整提问深度，收集故事与思辨
3. **Phase 3：结构与升华** — 提炼核心观点和面向未来的宣言
4. **Phase 4：图纸输出** — 生成结构化的演讲骨架，包含开场钩子、个人故事、洞察、升华

**特点：**
- 严格执行"一问一答"机制，绝不一次性抛出多个问题
- 只摘录用户的"原话"作为金句，严禁 AI 编造华丽辞藻
- 保持中立探究，不使用"太棒了"等主观评价词

### 模式B：演讲技巧优化模式（修改已有演讲稿）

当用户提供已有演讲稿时，从专业演讲教练的角度提供：
- 开场钩子分析（是否制造悬念、抓住注意力）
- 故事完整性评估（场景、细节、情感）
- 思辨深度检查（独特洞察、认知冲突）
- 金句提炼（从原文提取，保持真实感）
- 结尾升华建议（面向未来的呼吁）

输出完整的诊断报告和优化版演讲稿。

## 使用方法

### 安装

将 `youth-aited-mentor` 文件夹复制到你的 Claude Code skills 目录：

```bash
# Windows
cp -r youth-aited-mentor ~/.claude/skills/

# macOS/Linux
cp -r youth-aited-mentor ~/.claude/skills/
```

### 触发方式

在 Claude Code 中输入以下任一方式触发：

- `/aited`
- "TED演讲准备"
- "青少年演讲"
- "AI演讲辅导"

### 使用示例

**示例 1：从零开始准备演讲**
```
用户: /aited
AI: 嗨！我是你的TED思维探险伙伴。我想帮你找到一个只有你能讲的、独一无二的 AI 故事。

不用紧张，我们先随便聊聊——你今年多大了？你还记得自己第一次真正接触或使用 AI 是什么时候吗？
```

**示例 2：优化已有演讲稿**
```
用户: 帮我看看这个演讲稿：

大家好，我是小明。今天我想和大家分享AI如何改变了我的学习方式...

AI: [提供演讲稿诊断报告，包含开场、故事、金句、结尾的具体优化建议]
```

## 演讲力公式

$$
\text{演讲力} = \text{个人叙事(故事)} \times \text{认知冲突(思辨)} + \text{技术洞察(素养)}
$$

## 适用场景

- 青少年准备学校的 AI 主题演讲比赛
- TED 风格的演讲训练
- 需要从个人经历中提炼观点的演讲准备
- 已有演讲稿需要专业反馈和优化

## 不适用场景

- 成人演讲准备（本 skill 专为 10-17 岁青少年设计）
- 非 AI 主题的演讲（虽然方法论可迁移，但提问设计针对 AI 主题）
- 纯粹的演讲稿代写（本 skill 拒绝代写，只做引导和优化）

## 技术细节

- **状态机锁定**：严格的 4 阶段流程，防止跳跃式提问
- **年龄自适应**：根据用户年龄（10-12/13-15/16-17）调整提问深度和复杂度
- **原话提取**：金句必须从用户原话中提炼，保持真实性和年龄特征
- **双模式切换**：自动识别用户是需要引导（模式A）还是优化（模式B）
